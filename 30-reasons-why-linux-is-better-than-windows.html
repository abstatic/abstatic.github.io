30 Reasons Why Linux Is Better Than Windows

Sat 15 June 2013

Have you ever wondered why so many devs prefer linux over windows , or What exactly makes linux  better than other operating systems ?

Take a look  below to know why.

  1. Its free ! Contrary to windows and other OS linux is completely free, you can download it off the  net and install it, share it and that, my friend, is perfectly legal, no headaches about licensing.

  2. No viruses or spywares. Even Bill Gates cannot cure windows of its viruses, but linux on other hand rarely gets infected with viruses or spywares.Tight security is a core functionality in a linux PC and the number of viruses in linux is astronomically lower than windows.

  3. Say goodbye to disk fragmentation. Linux’s filesystem works very efficiently. Unlike FAT or NTFS it stores data in middle of a HDD sector, so you can append to any file as you please. Without worrying about disk fragmentation.

  4. It DOES NOT crash without any apparent reasons because the GUI(pronounced gew-ee) and the core operating system(the kernel) are completely independent and do not affect each other.

  5. It doesn’t require re-installation.unless there is a HDD failure. By creating a separate /home partition user data can be preserved even in case of reinstallation.

  6. It doesn’t require frequent re-booting even if you change settings or install any application.

  7. Linux can read over 1000 filesystems.

  8. Its open source , so you can modify it the way you want and maybe release your own distro.

  9. The shell prompt of linux known as BASH is very powerful and entire system administration can be done using it.

  10. Unlike windows you can install it on any type of partition of your HDD be it primary or logical.

  11. No one can monitor your activitieson linux unless you want them to , did you know all versions of windows come preloaded with a backdoor so that the US NSA can monitor your activities and data.

  12. Linux doesn’t use registry, hence no registry corruption! Most of the configuration is stored on plain text files

  13. You can even run linux from a LIVE CD , without ever having the need to install it permanently on your hard drive.

  14. Contrary to popular belief linux supports insane amount of hardwares , in windows you have to install the drivers for almost everything.

  15. The user support in linux is very wide and the online forums are extremely active , once you get started **its like one huge family.

  16. Moreover being open source it has very less number of bugs , and if any bug does occur it is fixed within HOURS.

  17. No frequent hardware upgrades are required in case of linux.It can run happily on older hardware and the hardware requirements don’t increase with every new release.

  18. Light On System Resources. Linux is super light on your system resources and you can utilize the full power of your machine.

  19. Best Development Platform. Most of the compilers are built-in, you don’t need to install anything extra.

  20. Excellent package management system. You can install all softwares in just a few clicks , that too for free and moreover they are very easy to upgrade.

  21. The default graphics are real eye candy something windows can never acheive.The desktop effects are simply amazing!

  22. Multiple Desktops.This is obviously one of my fav. parts of linux. Ever linux distro has multiple desktops which makes it very easy to multitask. Multiple desktops make it easy to have many programs open at once but provides system resources only to the ones you are currently using.

  23. Linux is highly customisableyou can use different window managers like Gnome , KDE , Unity
    You can even define your own custom shortcuts!  

  24. You are in total control of your system.There are no hidden applications/processes. You can   always decide which process will start when the kernel starts.

  25. With the help of clients like Empathy and gwibber you can **satisfy all of your social networking needs right from your desktop.

  26. All linux softwares are generally written in C therefore they are considerably more stable and powerful as compared to other softwares.

  27. Every distro has its own included appstore which provides you access to thousands of verified softwares and the silver lining is its all free!

  28. Browsing is not only more secure but considerably faster on linux. It also saves a lot of bandwidth.

  29. Linux plays friends with other OS. It recognizes many OS and lets you access/edit their files with great flexibility.

  30. All servers and super computers are linux based. You can have any no. of processors , and literally any amount of hardware , linux will recognize it and put it to full use. Whereas windows just cannot scale up to that level
